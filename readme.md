Первый ~~блин комом~~ проект на джанго.

## Основные технологии
- `python 3.9`
- `django`
- `poetry`
- `postresSQL 13`

## Запуск

Запустить пока можно с использованием `Makefile`:

```$ make install```.

Скоро будет запуск через docker-compose.

## OpenAPI
Можно получить последнее описание `$ make schema` или смотреть в `openapi-schema.yml`.
